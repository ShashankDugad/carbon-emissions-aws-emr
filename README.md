# Carbon Emissions Violation Detection
Big Data project analyzing 225M+ air quality records using AWS EMR + Spark

## Tech Stack
- AWS EMR (Spark 3.5.6, Hadoop 3.4.1)
- JupyterHub
- ML: Random Forest (99.25% AUC)
- Region: Mumbai (ap-south-1)
